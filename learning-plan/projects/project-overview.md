# 项目总览（双岗位适配）

## 项目 A：统一接入网关 + 鉴权中心
- 能力点：长连接、鉴权、限流、熔断、可观测性。
- 游戏岗话术：高并发在线与低延迟体验保障。
- 后台岗话术：统一接入层架构设计与稳定性交付。

## 项目 B：任务调度与状态编排服务
- 能力点：状态机、幂等、重试补偿、异步化。
- 游戏岗话术：匹配/房间/对局生命周期与可靠性。
- 后台岗话术：核心业务编排、子模块独立交付。

## 项目 C：数据服务层（MySQL + Redis + MongoDB）
- 能力点：MySQL 调优、Redis 缓存一致性、Mongo 文档建模。
- 游戏岗话术：玩家与实时数据读写优化。
- 后台岗话术：核心数据链路性能与稳定性优化。
