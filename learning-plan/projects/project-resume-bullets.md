# 简历话术（可直接使用）

## 项目 A：网关 + 鉴权
- 游戏岗：负责长连接网关与鉴权模块，支持心跳保活、断线重连与消息路由。
- 游戏岗：针对高并发在线场景实现限流熔断，保障高峰期服务稳定与低延迟。
- 后台岗：独立设计统一接入层，落地鉴权、流控、灰度路由并完成上线。
- 后台岗：建立日志、指标、告警体系，提升故障定位效率与交付质量。

## 项目 B：调度 + 状态机
- 游戏岗：实现匹配/房间生命周期状态机，支持超时回收与失败重试。
- 游戏岗：通过异步队列与幂等机制提升任务创建成功率。
- 后台岗：负责核心任务调度子模块开发，覆盖分发、流转、补偿流程。
- 后台岗：产出 HLD/LLD 并按期交付，保障需求质量闭环。

## 项目 C：数据服务层
- 游戏岗：设计玩家与排行数据存储，使用 Redis 加速热点读写。
- 游戏岗：慢 SQL 分析 + 索引优化，降低高并发场景查询延迟。
- 后台岗：构建 MySQL + Redis + Mongo 协同方案，优化吞吐与稳定性。
- 后台岗：建立缓存一致性与降级策略，提升异常场景可用性。
