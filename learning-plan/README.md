# Go 后端求职学习计划（分目录版）

## 目录说明
- `weekly/`: 每周独立计划文档（Week 01 ~ Week 10，概念优先 + 每日目标）
- `projects/`: 项目方案与双岗位简历话术
- `templates/`: 需求、设计、压测模板
- `guides/`: 执行引导、每周检查清单、完成标准

## 快速开始
1. 先读 `projects/project-overview.md`，确认 3 个主项目。
2. 再读 `guides/execution-standard.md`，明确每日/每周完成标准。
3. 从 `weekly/week-01.md` 开始执行，每天打卡。
4. 每周结束按 `guides/weekly-checklist.md` 输出交付物。

## 核心目标
- 一套项目适配两类岗位：游戏服务器岗 + 核心后台岗。
- 每个项目都有可运行代码、测试、压测报告、设计文档。
