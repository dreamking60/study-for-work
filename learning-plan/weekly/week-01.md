# Week 01：Go 基础与工程化（详细打卡版）

目标：完成 Go 入门 + 工程化最小闭环，产出可运行 CLI v0.1。

## Day 1
- 学习：Go 环境、go mod、项目目录组织。
- 编码：初始化项目，创建 `cmd/app/main.go`。
- 验收：`go run ./cmd/app`、`go test ./...` 正常。

## Day 2
- 学习：结构体、方法、接口、切片/map。
- 编码：实现 `internal/config` 与 `internal/model`。
- 验收：主流程可调用模块并输出结果。

## Day 3
- 学习：error 处理、JSON、文件读取。
- 编码：读取配置并处理非法配置错误。
- 验收：错误可定位配置项，异常可控失败。

## Day 4
- 学习：日志与 flag 参数。
- 编码：实现 logger，支持 `-config/-env/-verbose`。
- 验收：日志字段完整，参数错误有提示。

## Day 5
- 学习：单测、表驱动、覆盖率。
- 编码：配置加载和核心逻辑单测。
- 验收：测试通过，核心包覆盖率建议 >= 60%。

## Day 6
- 学习：CLI 命令组织与模块解耦。
- 编码：完成 CLI v1，至少 2 个子命令。
- 验收：命令可重复执行，异常输入可控。

## Day 7
- 学习：周复盘方法。
- 编码：整理 README 与目录说明。
- 验收：新人 10 分钟内可跑通。

## 本周最低完成线
- 一个可运行 Go CLI。
- 有模块拆分、测试与 README。
- 形成 `weekly-review/week1.md`。
